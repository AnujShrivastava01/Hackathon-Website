=========================================================
          🌊 HACKOCEAN : HACKATHON PORTAL 🌊
=========================================================
An immersive, interactive, and beautifully designed platform 
for hosting and managing world-class hackathons.

---------------------------------------------------------
🎨 UI/UX & DESIGN THEME (The "HackOcean" Experience)
---------------------------------------------------------
• Underwater Themed Aesthetic: A captivating deep-sea vibe using rich ocean blues, teals, and fluid dark-mode elements to keep participants engaged.
• Interactive Animations: Powered by Framer Motion, elements float, reveal, and transition smoothly on scroll, mimicking the flow of water.
• Modern Glassmorphism: Semi-transparent frosted glass effects on cards, navigation, and modals create a deep, layered interface.
• Clean & Bold Layouts: Taking inspiration from clean modern design (like Stripe and Apple), ensuring crystal-clear readability and intuitive navigation.
• Mobile-First Responsiveness: A fluid grid system using Tailwind CSS ensures the experience is flawless on any device, from ultrawide monitors to smartphones.

---------------------------------------------------------
✨ KEY PLATFORM FEATURES
---------------------------------------------------------
[ For Participants – The Public End ]
• Dynamic Hero Section: A striking entry point with animated visual elements and direct 'Register Now' access.
• Agenda & Schedule: A vertical, timeline-style breakdown of the event, updating dynamically so hackers never miss a phase.
• Collapsible FAQs: Interactive accordion menus to quickly address common queries.
• Real-time Quick Links: Direct access to registration, contact details, and social channels.

[ For Organizers – The Secure Admin Dashboard ]
• Secure Authentication: JWT cookie-based login ensures that only authorized admins can edit the hackathon details.
• Event Control Center: Update the Hackathon name, descriptions, dates, and venue in real-time.
• Live Registration Link Manager: Swap out Google Forms or external registration links instantly without redeploying code.
• Content Management: Create, Read, Update, and Delete (CRUD) operations for the entire Agenda, Schedule timeline, and FAQs visually through the dashboard.

---------------------------------------------------------
🛠 THE TECH STACK (Built with the MERN Stack)
---------------------------------------------------------
• Frontend: 
   - React.js & Vite (Lightning-fast rendering and build times)
   - Tailwind CSS (Utility-first styling for complex, responsive designs)
   - Framer Motion (Advanced physics-based scrolling and micro-interactions)
   - Lucide Icons & React Hot Toast (Beautiful iconography and elegant notification popups)
   - Axios (Reliable API integration)

• Backend:
   - Node.js & Express.js (Robust, scalable server architecture)
   - JSON Web Tokens [JWT] & BcryptJS (Industrial-grade security for authentication)

• Database:
   - MongoDB & Mongoose (Flexible, document-oriented NoSQL database perfect for handling diverse event data)
